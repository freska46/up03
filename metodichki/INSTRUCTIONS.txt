1. ЧТО СОЗДАНО
--------------
Приложение состоит из 3 частей:
- База данных PostgreSQL (pgAdmin 4)
- Web API (ASP.NET Core 8)
- WPF приложение (настольное)

Структура проекта:
PakDzal_Games/
├── PakDzal_Games.slnx          - Общее решение (открывать в Visual Studio)
├── PakDzal_Games_API/           - Web API сервер
│   └── Controllers/           - API контроллеры (Users, Games, Sessions)
├── PakDzal_Games_WPF/          - Настольное приложение
│   └── Views/               - Диалоговые окна
└── create_database.sql       - Скрипт создания БД

База данных:
- 3 таблицы: Users, Games, Sessions
- 5 пользователей из Курска
- 5 игр
- 5 сессий


2. ЗАПУСК
---------
Способ 1 - BAT файлы (самый простой):
1. Дважды кликните по run_api.bat - запустит API
2. В новом окне дважды кликните по run_wpf.bat - запустит приложение

Способ 2 - Visual Studio:
1. Откройте PakDzal_Games.slnx
2. Нажмите F5 (или выберите несколько проектов для запуска)


3. ПОДКЛЮЧЕНИЕ
----------------
API запускается на порту 5222
- http://localhost:5222 - главная страница
- http://localhost:5222/swagger - документация API
- http://localhost:5222/api/users - пользователи
- http://localhost:5222/api/games - игры
- http://localhost:5222/api/sessions - сессии

WPF приложение подключается к API автоматически при запуске.


4. ВОЗМОЖНОСТИ
----------------
Вкладка "Пользователи":
- Просмотр списка пользователей
- Поиск по имени/email/городу
- Добавить нового пользователя
- Изменить данные пользователя
- Удалить пользователя (с подтверждением)
- Отчет по пользователям

Вкладка "Игры":
- Просмотр списка игр
- Поиск по названию/жанру
- Фильтр по доступности
- Добавить новую игру
- Изменить данные игры
- Удалить игру (с подтверждением)
- Отчет по играм

Вкладка "Сессии":
- Просмотр списка сессий
- Фильтр по пользователю/игре
- Добавить новую сессию
- Изменить данные сессии
- Удалить сессию (с подтверждением)
- Отчет по сессиям


5. API ENDPOINTS
--------------
GET    /api/users              - все пользователи
GET    /api/users/{id}          - пользователь по ID
POST   /api/users              - создать пользователя
PUT    /api/users/{id}          - обновить пользователя
DELETE /api/users/{id}          - удалить пользователя
GET    /api/users/report        - отчет

GET    /api/games             - все игры
GET    /api/games/{id}         - игра по ID
POST   /api/games             - создать игру
PUT    /api/games/{id}         - обновить игру
DELETE /api/games/{id}         - удалить игру
GET    /api/games/report       - отчет

GET    /api/sessions         - все сессии
GET    /api/sessions/{id}    - сессия по ID
POST   /api/sessions         - создать сессию
PUT    /api/sessions/{id}     - обновить сессию
DELETE /api/sessions/{id}     - удалить сессию
GET    /api/sessions/report  - отчет


6. ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ
--------------------------
- Проверьте что PostgreSQL запущен
- Проверьте что БД pakdzal_games существует
- Проверьте что таблицы созданы (в pgAdmin)
- Проверьте что API запущен (консоль не закрыта)
- Проверьте порт 5222 не занят другим приложением
